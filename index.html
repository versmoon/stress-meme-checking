🧩 PIVOT TABLE — PANDUAN ZONA

🔽 FILTERS
Menyaring seluruh dataset. Contoh: Periode, Region, Status.
↔️ COLUMNS
Jadi header kolom horizontal. Contoh: Bulan, Cabang.
↕️ ROWS
Jadi label baris. Contoh: Kategori, Nama Produk.
🔢 VALUES
Angka yang dihitung (SUM/COUNT/AVG). Contoh: Omset, Qty.

📊 VIZ DESIGN — 5 ATURAN EMAS

✓
Bar Chart = Perbandingan kategori. Horizontal jika label panjang atau >6 item.
✓
Line Chart = Tren waktu. Ideal untuk perubahan dari waktu ke waktu.
✓
Pie/Donut = Proporsi. Maksimal 5 irisan, selisih harus signifikan.
✗
Hindari: 3D chart (distorsi perspektif), warna pelangi (gangguan fokus), dual axis membingungkan.
★
Grey-Out Trick: Abu-abukan semua, highlight 1 data target pakai warna neon. Teknik eksekutif pro!
